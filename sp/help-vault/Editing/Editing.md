# Editing

StillPoint is a Markdown editor with live formatting and a fast, keyboard-friendly flow.

## Markdown Basics
- Headings with `#` and Ctrl+1..Ctrl+5.
- Bold, italic, strike, highlight, and inline code.
- Lists and checkboxes with automatic formatting.

## Quick Formatting Shortcuts
- Bold: Ctrl+B
- Italic: Ctrl+I
- Strikethrough: Ctrl+K
- Highlight: Ctrl+U
- Inline code: Ctrl+T
- Remove heading: Ctrl+7
- Clear formatting: Ctrl+9

## Example
**Bold**
*Italic*
~~Strikethrough~~
==Highlight==
`inline code`

## Checkboxes
- [ ] First task
- [x] Completed task

## Asterisk and Dash lists
